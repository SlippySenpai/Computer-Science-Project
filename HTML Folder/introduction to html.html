HyperText Markup Language or HTML

is the standard markup language for documents designed to be displayed in a web browser.
It defines the meaning and structure of web content.
Assisted by technologies such as cascading style sheets and scripting languages such as java script.

There are so many different tags and attributes in html that helps us to create the page we desire.

Examples of some of these are :-

  1. Img Src
  2. A Href
  3. Ordered List
  4. Unordered List
  5. Table Tags

A nd many more.

Using these tags we can create some of the most unique pages that one can imagine.

Now, Let's talk about different languages of web programming.

  • You can find this table at the bottom of the page

  • JavaScript :- JavaScript Helps in making the web pages interactive. It allows developers to create dynamically updating content, use animations, pop-up menus, clickable buttons, control multimedia etc.
  • Cascading Style Sheet (CSS) :- It uses a separate language from HTML. CSS Allows you to apply consistent styling of elements across all pages on your site, so that all headings, lists, and paragraphs look and act the same on every page of a site.
  • Structured Query Language (SQL) :- SQL is a database management system used to create and manage servers. With SQL programming language knowledge, you can easily navigate through massive amounts of data and understand how to retrieve data from a web application. Knowing SQL will allow you to extract information from website quickly.
  • Python :- Python Web Application development typically employs python framweworks to construct its backend. These tasks, including parsing HTTP requests and creating responses; accessing databases: authenticating users; authorizing access; etc.
